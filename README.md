# 🚀 fuyun_tools

[中文](README.md) | [English](README_EN.md)

fuyun_tools 是一款常驻系统托盘的桌面效率工具，聚焦两件事：

- 更高效地管理剪贴板历史
- 在 Windows 上进行 AI 划词翻译与解释

---

## ✨ 功能概览

### 📋 剪贴板管理

- 自动记录剪贴板历史，支持快速回填
- 支持搜索、分类、删除和历史容量上限配置
- 支持键盘与鼠标双路径操作（方向键、回车、滚轮等）

### 🔤 AI 划词助手（Windows）

- 支持拖拽、双击、三击等选词场景
- 自动弹出划词工具栏，可直接翻译/解释/复制
- 结果窗口支持流式输出，阅读反馈更及时

### 🤖 AI 服务配置

- 内置 DeepSeek / 通义千问 / 小米 Mimo
- 支持新增任意 OpenAI 兼容自定义提供商
- 支持在下拉选项中直接删除自定义提供商
- API Key 本地加密保存

### ⚙️ 系统集成

- 常驻托盘，支持开机自启
- 全局热键唤出剪贴板窗口
- 深浅色主题切换
- 内置应用更新能力

## 🚀 快速开始

1. 运行应用后，在系统托盘中找到图标
2. 使用默认热键唤出剪贴板窗口
    - Windows: `Ctrl+Shift+Z`
    - macOS: `Cmd+Shift+Z`
3. 在“设置 → AI 设置”中配置提供商、模型与 API Key
4. 在 Windows 任意应用中选中文本，使用划词工具栏进行翻译或解释

## 🧭 使用说明

### 剪贴板窗口

- `↑ / ↓`：上下选择
- `Enter`：回填当前项
- `Esc`：隐藏窗口
- 鼠标滚轮：滚动列表

### 划词工具栏（Windows）

- 选中文本后自动显示
- 点击翻译/解释后在结果窗口查看流式内容
- 点击工具栏外部区域自动收起

### AI 配置建议

- API 地址需以 `http://` 或 `https://` 开头
- 模型名称需与服务端可用模型一致
- 建议先使用“连接测试”再保存

## 🛠️ 本地开发

### 技术栈

- 前端：Vue 3 + Element Plus
- 桌面框架：Tauri 2 + Rust
- AI SDK：async-openai（OpenAI 兼容接口）

### 常用命令

前端构建：

```bash
cd src
npm run build
```

Tauri 检查：

```bash
cd src-tauri
cargo check
```

## ❓ 常见问题

### 1) 为什么 Linux/macOS 没有划词功能？

当前版本仅在 Windows 完成了划词链路实现，其他平台后续补齐。

### 2) 自定义提供商怎么删除？

在 AI 提供商下拉框中，对应自定义项右侧点击 `X` 即可删除。

### 3) 保存设置后关闭窗口偶发延迟？

已针对关闭流程做响应优化，建议更新到最新版本。
